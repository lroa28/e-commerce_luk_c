NUESTRA PRIMERA BASE DE DATOS EN MONGODB

Realizar las siguientes acciones utilizando un servidor de base de datos MongoDB:

Crear una carpeta llamada 'base' para contener la estructura de base de datos.
Ejecutar en una consola el motor de base de datos MongoDB, que debe seleccionar la carpeta 
'base' como almacenamiento de su estructura.
Iniciar Mongo shell en otra consola, verificando la conexión con el servidor.
Listar las bases de datos disponibles: 

/* Comando ejecutado: */
> show dbs
admin      0.000GB
config     0.000GB
ecommerce  0.000GB
local      0.000GB
sistema    0.000GB

Crear una base de datos en MongoDB llamada 'sistema' y seleccionarla con base de trabajo.

/* Comando ejecutado: */
> use sistema

Insertar un documento en la colección usuarios que contenga los campos: nombre y edad.

/* Comando ejecutado: */
> db.usuarios.insertOne ({nombre: 'Pepe', edad: 30})

Listar los documentos dentro de la colección usuarios.

/* Comando ejecutado: */
> db.usuarios.find()

Mostrar las colecciones de la base 'sistema'.

/* Comando ejecutado: */
> use sistema
> show collections
sistema
usuarios

Insertar tres documentos más con el mismo formato y valores distintos en la colección usuarios.

/* Comando ejecutado: */
> db.usuarios.insertMany ([{nombre: 'Alicia', edad: 10}, {nombre: 'Horacio', edad: 40}, {nombre: 'Brad', edad: 25]);

Ejemplo desde https://www.mongodbtutorial.org/mongodb-crud/mongodb-insertmany/
db.books.insertMany([
   { title:  "NoSQL Distilled", isbn: "0-4696-7030-4"},
   { title:  "NoSQL in 7 Days", isbn: "0-4086-6859-8"},
   { title:  "NoSQL Database", isbn: "0-2504-6932-4"},
]);


Listarlos nuevamente verificando que haya 4 documentos dentro de la colección usuarios.

/* Comando ejecutado: */
> db.usuarios.find()

Insertar dentro de una colección llamada 'productos' cuatro documentos con los siguientes campos: nombre, 
precio y stock.

/* Comando ejecutado: */
> use productos
> db.productos.insertMany ([{nombre: 'heladera', precio: 1000, stock: 100}, {nombre: 'cocina', precio: 2000, stock: 100}, {nombre: 'lavarropas', precio: 3000, stock: 100}, 
{nombre: 'frizzer', precio: 4000, stock: 100}]);

Mostrar las colecciones de la base 'sistema'.

/* Comando ejecutado: */
> use sistema
> show collections

Listar todos los documentos de la colección 'productos'.

/* Comando ejecutado: */
> use productos
> db.productos.find()
